# Covid-19-Visualization

### OVERVIEW
simple script to visualize the moving averages (5,10) days of confirmed Covid 19 cases in Oman. The script fetches the data from the endpoint https://albahri.pythonanywhere.com/statsbydate

### HOW TO RUN THE CODE
install the requirements
```
pip install -r requirements.txt
```
then run the script using python.
